# Simple opengl 'bench' in C
This project uses:  
### Linux
 * [freeglut](http://freeglut.sourceforge.net/)

### All platforms 
 * [cmake](https://cmake.org/)


## Building
### Linux
Needs ***freglut*** installed on your system, on ubuntu, something like:
```bash
sudo apt install freeglut3-dev
```
and also cmake
```bash
sudo apt install cmake
```
